# Superstore Sales Analysis Dashboard (Power BI)

## Project Overview
This project analyzes Superstore sales data using Microsoft Power BI.

## Tools Used
- Power BI
- Excel/CSV
- DAX

## Dashboard Features
- Sales and Profit KPI Cards
- Monthly Sales Trend
- Sales by Category and Sub-Category
- Profit by Region
- Top 10 Products Analysis
- Interactive Filters

## Key Insights
- Identified top-performing categories and products
- Analyzed regional profitability
- Calculated profit margin for business performance tracking
